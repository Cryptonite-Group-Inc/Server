# NFT Worlds Server
The open-sourced server for NFT Worlds. 

This repo is a work in progress, official release coming soon.

[Homepage][home] - [Discord][discord] - [About NFT Worlds][about]

## Building

```bash
./gradlew build
```

Or building a Paperclip JAR for distribution:

```bash
./gradlew paperclip
```
